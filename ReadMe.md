# ZKA StageBoard.

A Mobile API and Launcher written in C++.

## Installing:

Start by cloning the repo.

```
git clone git@bitbucket.org:mahrouss/priv-dll.git
```

## Building:

Use __btb__ to build the frameworks.

```
btb launcher.json
```

You can now use the DLLs.
