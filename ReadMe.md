# ZKA AppLauncher.

An Mobile app launcher written in C++, with it's own API.

## Installing:

Start by cloning the repo.

```
git clone git@bitbucket.org:mahrouss/priv-dll.git
```

## Building:

Use btb to build the frameworks.

```
btb build.json
```

You can now use the DLLs.
