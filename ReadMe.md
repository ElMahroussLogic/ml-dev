# New OS standard DLL.

## Installing:

Start by cloning the repo.

```
git clone git@bitbucket.org:mahrouss/dll.git
```

## Running makefiles:

And then run the makefile according to your directory.

```
make all
# You can use that also.
btb build.json
```

You can now use the programs.
